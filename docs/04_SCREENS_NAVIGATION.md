# 화면 설계 및 네비게이션 플로우 문서

## 1. 네비게이션 구조

```
App Root (_layout.tsx)
  │
  ├─ 인증 체크: 미로그인 → (auth) 그룹
  │                로그인 → 역할 체크
  │                         ├─ instructor → (instructor) 그룹
  │                         └─ member → (member) 그룹
  │
  ├── (auth)/
  │   ├── login.tsx
  │   └── register.tsx
  │
  ├── (instructor)/              ← 하단 탭 네비게이터 (5탭)
  │   ├── dashboard.tsx          [Tab 1: 대시보드]
  │   ├── members/               [Tab 2: 회원]
  │   │   ├── index.tsx
  │   │   └── [id].tsx
  │   ├── schedule.tsx           [Tab 3: 스케줄]
  │   ├── sessions.tsx           [Tab 4: 세션]
  │   └── settings.tsx           [Tab 5: 설정]
  │
  │   ※ 모달/스택 화면:
  │   ├── review.tsx             (시퀀스 리뷰)
  │   └── member-form.tsx        (회원 등록/편집)
  │
  └── (member)/                  ← 하단 탭 네비게이터 (4탭)
      ├── today.tsx              [Tab 1: 오늘]
      ├── condition-check.tsx    [Tab 2: 컨디션]
      ├── history.tsx            [Tab 3: 기록]
      └── profile.tsx            [Tab 4: 프로필]
      │
      ※ 모달/스택 화면:
      └── exercise-detail.tsx    (운동 상세)
```

## 3. 화면 전환 플로우

### 3.1 회원: 컨디션 체크 → 시퀀스 생성

```
[오늘 탭]
  │ "컨디션 체크하기" 탭
  ▼
[컨디션 체크 탭: 카메라]
  │ 촬영 버튼 탭
  ▼
[API: POST /condition/analyze]
  │ 로딩 (5초 이내)
  ▼
[분석 결과 표시]
  │ 에너지/무드/스트레스/수면/근긴장/부종 표시
  │
  │ (선택) 슬라이더/선택지로 수정
  │ (선택) 추가 메모 작성
  │ (선택) 추가 운동 카테고리 선택
  │
  │ "등록 & 시퀀스 생성" 탭
  ▼
[API: POST /condition/register]
  │ AI 원본 + 수정본 + 메모 + 카테고리 저장
  ▼
[API: POST /sequences/generate]
  │ 로딩 (10초, 프로그레스)
  ▼
[완료 토스트: "오늘의 시퀀스가 생성되었습니다!"]
  │ 오늘 탭으로 자동 이동
```

### 3.2 강사: 스케줄 관리

```
[스케줄 탭]
  │ 요일 선택
  ▼
[해당 요일 스케줄 목록]
  │ [+ 스케줄 추가] 탭
  ▼
[스케줄 등록 모달]
  │ 회원 선택 + 시간 설정
  │ "저장" 탭
  ▼
[API: POST /api/schedules]
```

### 3.3 알림 + 자동 생성 (백그라운드)

```
[12:00 크론잡]
  │ 오늘 스케줄 중 컨디션 미체크 회원 조회
  │ push 알림 발송
  ▼
[13:00 크론잡]
  │ 여전히 미체크 회원 대상
  │ 자동 시퀀스 생성 + 배정
  │ push 알림 + 대시보드 알림
```

## 4. 공통 UI 컴포넌트

| 컴포넌트 | 용도 | 사용 화면 |
|---|---|---|
| `FaceCapture` | 카메라 프리뷰 + 캡처 | 컨디션 체크 |
| `ConditionResult` | AI 분석 결과 표시 | 컨디션 체크, 대시보드 |
| `ConditionEditor` | 컨디션 수정 UI (슬라이더/선택지) | 컨디션 체크 |
| `ConditionBadge` | 에너지/무드/스트레스 요약 뱃지 | 대시보드, 히스토리, 리뷰 |
| `CategorySelector` | 추가 운동 카테고리 선택 | 컨디션 체크 |
| `SequenceCard` | 시퀀스 전체 요약 | 리뷰, 오늘의 시퀀스 |
| `ExerciseItem` | 개별 운동 카드 | 리뷰, 오늘의 시퀀스 |
| `SequenceEditor` | 드래그 앤 드롭 편집 | 리뷰 (강사) |
| `WeeklySchedule` | 주간 스케줄 뷰 | 스케줄 (강사) |
| `ScheduleEditor` | 스케줄 편집 | 스케줄 (강사) |
| `SessionStatusCard` | 세션 현황 카드 | 대시보드 (강사) |
| `MemberCard` | 회원 목록 아이템 | 회원 목록 |
| `BodyConditionTag` | 신체 상태 태그 | 회원 상세, 프로필 |
| `Slider` | 컨디션 수정용 슬라이더 | 컨디션 체크 |
